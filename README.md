# Student Management System

## Overview
A simple Java console application for managing student information.  
The project demonstrates **OOP**, **input validation**, **exception handling**, and **file I/O**.

---

## Features
- Add student with validation  
- Prevent duplicate student IDs  
- Display students in formatted table  
- Search by partial name  
- Sort by first name or GPA (descending)  
- Delete student by ID  
- Save/load data from file  

---

## Technologies
- Java Core  
- OOP  
- Collections (ArrayList)  
- File I/O  
- Regex validation  

---

## Validation Rules
- **ID**: `S + 3 digits` (e.g. `S001`)  
- **Name**: max 30 characters  
- **Age**: 0 – 120  
- **GPA**: 0.0 – 4.0  

---

## Run
1. Run `Main.java`
2. Use menu to manage students
3. Enter `0` to exit
